`Libft` is the first project at **school 42**. 

The goal is to create a personal C library by re-coding a selection of standard C library functions (like `ft_memcpy`, `ft_strlen`, `ft_atoi`) as well as additional utility functions that will be used in future 42 projects.

<br/>
<br/>
<br/>

### How to Use

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/iTsLhaj/Libft.git](https://github.com/iTsLhaj/Libft.git)
    ```
2.  **Compile the library:**
    ```bash
    cd Libft
    make
    ```
    This creates the static library file `libft.a`.

3.  **Link it to your project:**
    ```bash
    cc your_project.c -L. -lft -o your_program
    ```
